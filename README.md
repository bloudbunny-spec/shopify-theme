## 安装与使用

### 主题安装
1. 登录您的 Shopify 管理后台
2. 进入"在线商店" > "主题"
3. 点击"上传主题"
4. 上传主题的 .zip 文件
5. 发布主题或预览

### 主题定制
1. 在 Shopify 管理后台中，进入"在线商店" > "主题"
2. 点击"自定义"按钮
3. 使用主题编辑器调整颜色、字体和布局
4. 根据需要启用或禁用各种功能
5. 保存您的更改
